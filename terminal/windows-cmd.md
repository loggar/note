# windows cmd

## Create a file

```
$ type nul > file1.txt
```

```
$ echo some text > file3.txt
```

appending text to a file

```
$ echo some text >> file3.txt
```

## Create a directory

```
$ mkdir dirname
```

## Delete a file

```
$ del hello.txt
```

## Delete a directory

```
$ rmdir /s dirname

$ rmdir /s "if have spaces"
```
