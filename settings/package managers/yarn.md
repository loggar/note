
[https://yarnpkg.com/en/docs/usage](https://yarnpkg.com/en/docs/usage)

install yarn

```
npm install -g yarn
```

install package.json

```
yarn init
```

install packages

```
yarn install
```

add dependencies

```
yarn add <package>
```

add devDependencies

```
yarn add -D <package>
```

remove dependencies

```
yarn remove <package>

yarn remove -D <package>
```
