# WARP

https://app.warp.dev

## Shell

```
echo $SHELL
chsh -s $(which zsh)
source ~/.zshrc
```

## Keyboard Shortcuts

https://docs.warp.dev/features/keyboard-shortcuts

- Remove everything in the current command: command mode
- `*`: Natural language mode
- `#` or Ctrl + `Backtic`: ASK suggestions
- Ctrl + R: Search
