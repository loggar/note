# macOS python

## alias

```
$ brew info python

$ alias python=/opt/homebrew/bin/python3
```

## pyenv

```
$ brew install pyenv

# $ pyenv init
# ~/.zshrc, or ~/.profile or ~/.bashrc
# eval "$(pyenv init -)"
```

```
$ pyenv install 3.10.7
```

```
$ pyenv versions
$ pyenv global 3.10.7
$ python --version
```

```
$ pyenv uninstall 3.10.7
```
