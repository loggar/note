# goland

## keymap

```
Go back: cmd + [
Go forward: cmd + ]

Find: cmd + f
Find in files: cmd + shift + f

Navigate to file: double shift
Navigate to Type: cmd + o
Navigate to File: cmd + shift + o
```

```
Debug - execute next line: F8
Debug - view all breakpoint: cmd + shift + F8
```

```
Increase font size: ctrl + shift + .
Decrease font size: ctrl + shift + ,
```

```
Delete a line: cmd + backspace
```

```
Refactor this: ctrl + t
```

```
Move line up:   opt + shift + up
Move line down: opt + shift + down
```
