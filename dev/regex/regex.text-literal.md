1. 문자 그대로 검색

"car"라는 단어 역시 정규 표현식이며, 평범한 텍스트도 정규 표현식이 될 수 있다.

[예문]
My car is SM3 manufactured in 2003

[정규 표현식]
car

[결과]
My car is SM3 manufactured in 2003

역시 평범한 단어로 검색하는 또 하나의 예제를 살펴보자.

[예문]
I'm a boy.
You're a girl.
Girls like boys.

[정규 표현식]
boy

[결과]
I'm a boy.
You're a girl.
Girls like boys.

예문에 2개의 "boy"가 포함되어 있었고, 정규 표현식 "boy"로 검색시 2개의 "boy"가 검색되었다.
두번째 결과는 "boy"가 포함된 "boys"이지, 정확하게 매칭되는 녀석은 아니다.

정확하게 내가 원하는 결과를 매칭시키는 것은 차차 알아가기로 하고,
이 예제에서는 복수의 매칭 결과를 얻을 수 있다는 정도를 이해하면 될 듯 하다.


2. 모든 문자 검색

앞서 문자 또는 문자열을 그대로 사용해서 매칭시키는 것을 살펴 보았다.
단지, 정규 표현식으로 문자 또는 문자열 그대로 검색도 가능하다는 것을 보여주려고 한 것이지, 정규 표현식의 매력이라곤 할 수 없다.

이제부터 슬슬 메타 문자(정규 표현식 안에서 특수한 의미를 가지는 문자)들을 활용하여, 검색하는 것을 살펴 보기로 하자.

그 첫걸음은 뉴라인 문자(\n)를 제외한 아무 문자 하나를 검색할 수 있는 .(마침표)이다.

[예문]
cat cal cu. cut cam

[정규 표현식]
c.t

[결과]
cat cal cu. cut cam

정규 표현식 "c.t"는 "c" 그리고 "아무 문자" 그리고 "t"로 구성된 단어를 의미한다.

메타 문자 .(마침표)는 뉴라인 문자를 제외한 아무 문자 하나를 검색할 수 있다고 하였으니, 자기 자신인 .(마침표)도 이에 해당한다.

[예문]
cat cal cu. cut cam

[정규 표현식]
cu.

[결과]
cat cal cu. cut cam

하나의 정규 표현식에서 메타 문자 .(마침표)는 여러 개가 함께 사용될 수 있다.
연달아 여러 개를 붙일 수도 있고,

[예문]
cat cal cum cut cam

[정규 표현식]
c..

[결과]
cat cal cu. cut cam

여러 개를 원하는 위치에 둘 수도 있다.

[예문]
cat cal cu. cut cam

[정규 표현식]
.a.

[결과]
cat cal cu. cut cam


3. 특수 문자 검색

메타 문자 .(마침표)는 뉴라인 문자를 제외한 아무 문자 하나를 일치시킬 수 있다고 하였다.
그럼 콕 집어서 .(마침표)를 검색하고 싶을 땐 어떻게 해야 할까?

[예문]
cat cal cu. cut cam

[정규 표현식]
c.\.

[결과]
cat cal cu. cut cam

C나 C++ 일반에서 특수 문자 표기시 이스케이프 시퀀스를 사용해야 한다는 것을 잘 알고 있을 것이다.
(물론 C++11의 Raw string literal을 사용하면 얘기가 조금 다르긴 하다만...)

정규 표현식에서의 이스케이프 시퀀스는 딱 1개, \(역슬래쉬) 뿐이다.
여러 특수 문자들 중 어떤 녀석들은 \(역슬래쉬)를 붙이지 않아도, 해당 문자 그대로 인식되는 경우가 있긴 하나,
어떤 건 되고, 어떤 건 안되니 이들을 따로 외워서 기억하기 보다는 모든 특문에 이스케이프 시퀀스를 붙이는 습관이 더 좋다 하겠다.

따지고 보면, \(역슬래쉬) 역시 특수(메타) 문자들을 문자 그대로 해석되게 해주는 메타 문자라 할 수 있다.
그리고, \(역슬래쉬) 역시 문자 그대로 해석되게 하려면 스스로를 이스케이프 시켜야 한다.


4. 정리

정규 표현식은 문자들로 이루어진 문자열이라 할 수 있다.
이러한 문자들은 그냥 의미 그대로의 문자일 수도 있고, 메타 문자일 수도 있다.
문자 그대로 검색할 수 있고,
.(마침표)는 뉴라인 문자를 제외한 아무 문자 하나를 검색할 수 있다.
\(역슬래쉬)는 특수 문자가 문자 그대로 해석되게 한다. (이스케이프)
