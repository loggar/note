```
기본 메타 문자		
	.	모든 문자
	|	왼쪽 혹은 오른쪽 (OR)
	[]	문자 집합 구성원 중 하나
	[^]	문자 집합 구성원이 아닌것
	-	범위 정의
	\	Escape
		
수량자		
	*	0회 이상 반복
	*?	게으른 * 문자
	+	1회 이상 반복
	+?	게으른 + 문자
	?	0회 혹은 1회 등장
	{n}	n회 반복
	{m,n}	m에서 n회 반복
	{n,}	n회 이상 반복
	{n,}?	게으른 {n,}
		
게으른
	최소단위를 찾는다	
	1. <b>.*</b>	탐욕적	일치하는 가장 큰 단위의 문자열을 탐색
	2. <b>.*?</b>	게으른	일차히는 가장 작은 단위의 모든 문자열을 분할 탐색
	ex)	<b>SOMETHING!</b><br /><b>ELSE</b>	1개의 탐욕적 일치 결과와 2개의 게으른 일치결과
		
		
위치 지정		
	^	문자열의 시작
	\A	문자열의 시작
	$	문자열의 끝
	\Z	문자열의 끝
	\<	단어의 시작
	\>	단어의 끝
	\b	단어의 경계
	\B 	\b 가 아닌것
		
특수한 문자		
	[\b]	backspace
	\c	제어문자
	\d	숫자
	\D 	\d 가 아닌것
	\f	페이지 넘기기 (form feed)
	\n	new line
	\r	carriage return
	\s	space
	\S 	space 가 아닌것
	\t	tab 문자
	\v	수직 tab
	\w	영어 문자, 숫자, 언더스코어(_) 중 하나
	\x	16진수 숫자
	\O	8진수 숫자
		
역참조, 저후방 탐색		
	()	하위 표현식 정의
	\1	첫번째 일치한 하위 표현식
	\2	두번째 일치한 하위 표현식
	?=	전방 탐색
	?<=	후방 탐색
	?!	부정형 전방 탐색
	?<!	부정형 후방 탐색
	?(breakference)true	조건 지정
	?(breakference)true|false	else 표현식 조건 지정
		
flag		
	/i	not case sensitive
	/g	global
	/m	Multi-line
```
