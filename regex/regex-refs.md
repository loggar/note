phone number

```
/^[+]*[(]{0,1}[0-9]{1,4}[)]{0,1}[-\s\./0-9]{6,16}$/
```

email

```
/^(([^<>()\[\]\\.,;:\s@"]+(\.[^<>()\[\]\\.,;:\s@"]+)*)|(".+"))@((\[[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}])|(([a-zA-Z\-0-9]+\.)+[a-zA-Z]{2,}))$/
```

no-1-space after comma

```
[,]{1}[^\s|"]{1}
```

person name

```
^[a-zA-Z]+(([a-zA-Z, \'\` \.\-])?[a-zA-Z]*)*$
```
