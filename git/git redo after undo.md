## Recovering lost commits

```
#make changes to HEAD
git commit -am "some WIP changes"
```

proceed undo it

```
git rebase -i origin/master
```

want redo it, show reflog

```
git reflog
```

recover it with HEAD-ID

```
git reset HEAD@{10}
```
